GOOGLE SHEETS LIVE DATABASE INTEGRATION

Connect your tournament to a live Google Sheet to share registrations and scores instantly across all players and devices.

🔍 HOW DO I SET THIS UP FOR FREE? (STEP-BY-STEP) ▼
1. Create a Google Sheet and name your columns in row 1: id, playerId, inGameName, discordTag, avatarUrl, score, createdAt.
2. Click Extensions → Apps Script. Paste the provided backend code.
3. Click Deploy → New Deployment → Web App. Set Access to "Anyone". Copy the Web App URL.
4. Paste the URL here to immediately sync registrations and leaderboard scores across the globe!
